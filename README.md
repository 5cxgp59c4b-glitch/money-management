# 支出管理アプリ

個人用の支出管理Webアプリです。

## ⚠️ 注意事項

- このアプリのコードは **AI（Claude）によって生成** されたものです
- 作成者はコードの詳細を完全には理解していません
- **私的利用のみ**を目的としています

## 機能

- カテゴリの追加・削除（色分け対応）
- 支出の記録（日付・カテゴリ・金額・メモ）
- カテゴリ別割合の円グラフ
- 月別支出推移の折れ線グラフ
- データのエクスポート（JSON / CSV）
- データのインポート（JSON で完全復元 / CSV で追加）

## データの保存先

ブラウザの **LocalStorage** に保存されます。外部サーバーへの送信は一切行いません。

## 技術スタック

- React + TypeScript
- Vite
- Recharts（グラフ）
